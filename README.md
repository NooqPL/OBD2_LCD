# OBD2_LCD
LCD display for mx5 nc connected to the car with OBD 2 car


set up your pi
name `mx5` 
domain if you want website `mx5` and the website is `mx5.local:5000`

after login in just run this command:
```
curl -sSL https://raw.githubusercontent.com/NooqPL/OBD2_LCD/main/install.sh | bash
```
